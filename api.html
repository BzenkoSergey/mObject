Model

Model - is the object that contains some data and behavior with possibility to pass them to child.

Basic Concepts

  • Model has included complex system of inheritance and configuration of creating new instances or childs.
  • model is a instance of the Model and gets all data/behavior from the Model, with some features that are cloned from Model to this instance for personal using and changing in future.

init()

Init is one unit of initialization instance, that has full control on the process and can stop the process that leads to destruction of the instance.

Is usually used for adding some functionality or break instance cases.

Is not available to manual calls and executes on instance creation or re-init.
  • arguments: data from instance creating
  • scope execution: current model instance
  • return: in case of false the instance creation/re-init will be broken. (only false value is processed)

inits stack

Inits stack - is the list of init and is as part of configuration instance creation. The list executes during instance creation or re-init, in case of anyone init returns false, the inits iteration will be stopped and instance will not be created.

Each Model/model contains its own inits stack that can be modified without negative affecting on other Models/models.

The list is mutable and inheritable and can be modified via API extendInit()

IIFE Mode

This mode means that passed callback will be executed and its result will be placed in the Model/model by key name. In case when IIFE callback returns undefined the result and key name will be ignored.

This is inheritable part and the execution process will be repeated on new Model child creation.

Child Descriptor

Descriptor contains some features (methods, props) that will be applied to child. The descriptor has ability to add some configuration params such as init, abstract, inheritable.

abstract

It prevents the creation instances from the Model. When you try to create instance, an exception will be thrown!
default: false

inheritable

Allows creation child from Model.
default: true

isModel

The property means that this object is Model constructor.
default for Model constructor: true

uniqueKey

Unique key used for models tracking.
Models collection avoids the creation of a new model with the same value of unique key.
default: id

extend()

Creates a new child with the features according to descriptor.
return: created child

extendInit()

Adds one init callback to inits stack.
default for Model constructor: true

extendClass()

Extend existing Model constructor.
All created childs of the Model will have these added specifics.

extendProto()

Extend existing Model prototype that is injected to all model instances.

reInit()

Re-run all inits stack.
